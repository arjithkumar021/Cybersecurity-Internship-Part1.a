# Cybersecurity Internship - Part 1

This repository contains the documentation, reports, and evidence for Part 1 of the Cybersecurity Internship.

## Structure
- **Screenshots/**: Evidence of work (Kali login, terminal usage, etc.)
- **Lab-Setup/**: Documentation on how the lab was set up.
- **CheatSheets/**: Quick reference guides created during the internship.
- **Notes/**: Daily notes and learning logs.
- **Reports/**: Formal reports submitted.
- **Video/**: Video demonstrations.
